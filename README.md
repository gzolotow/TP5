# TP5
Ejecutar

En la terminal:

node src/index.js

Después abrir el navegador en:

http://localhost:3000

 Rutas
/ - muestra un mensaje de bienvenida
/Hora - muestra la hora actual
/fecha-completa - muestra la fecha completa

 Error 404

Si entrás a una ruta que no existe, muestra un error 404.

 Extra

Agregamos HTTP Cats para que en el error 404 aparezca una imagen de un gato.

 Estructura

src/

app.js - rutas
index.js - levanta el servidor
time.js - función de la hora
